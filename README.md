# probe-localdns

This project demonstrates how to do getting local DNS address by http server.

## Deploy Project

### DNS Settings

Specify a `domain` name ns record to deploy probe-localdns server address.

### Install dependencies

```bash


npm install

```

### Running

```bash

npm run server <domain> <ip>

```
